Entorno de desarrollo 

- Node.js
- Editor: Sublime
- Plugin para typescript

Una vez instalado node.js, se abre un prompt y se escribe node -v para comprobar todo ok

Para instalar el plugin en sublime, necesitamos instalar package control y luego:
CNTROL+SHIFT+P > Package Control: install package > Typescript
Esto nos proporciona autocompletado y error checking

En el terminl de node.js, escribmos 
install typescript typings -g

La g indica ue guarde una copia local de cada libreria.
